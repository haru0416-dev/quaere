Quaere /ˈkwae̯.re/ — "ask"

Process-correction skills that make coding agents ask before they act.

Install

curl -fsSL https://quaere.dev/install.sh | sh

Or via Rust: cargo install quaere-cli && quaere install.

Skills

SkillUse when
quaere-semanticUnderstand unfamiliar code before changing it.
quaere-groundingVerify external, version-sensitive facts (SDKs, APIs, advisories).
quaere-evidenceDefend or reject claims, bug causes, review comments.
quaere-executionApply a plan with diff review and commit discipline.
quaere-auditProperty-driven security audit with exploitability gates.

Manage

quaere list      # show installed skills
quaere doctor    # verify install integrity
quaere update    # check for a newer release
quaere version   # print version
